folder CASE-2023-0492 / HARDWARE / DISTRICT 7

Smart Controller RF Signal Interference

report_problem

Triệu chứng

Recurrent communication timeouts observed on Segment 42-A during peak hours (06:00 - 09:00). Field technicians report "No Link" status on handheld diagnostics despite line-of-sight to gateway. Error logs indicate packet loss exceeding 45% with high noise floor readings (> -85dBm).

psychology

Root Cause

Interference identified as harmonic noise from newly installed 5G small cell infrastructure on adjacent private property. The antenna array's proximity (approx. 12 meters) to our 900MHz mesh controllers causes significant front-end overload on the receiver circuit, desensitizing the LNA (Low Noise Amplifier).

build

Repair Action

  • Installation of high-rejection 900MHz bandpass filters (Part #KMS-RF-BPF-900) on all gateway-facing controllers.
  • Orientation adjustment of the directional Yagi antennas to minimize sidelobe pickup from the 5G tower.
  • Relocation of the Segment Gateway (G-14) to a higher elevation point (+4m) on the existing utility pole.
lightbulb

Lessons Learned

"Future site surveys must include a spectrum analysis for 5G harmonic overlaps, especially in mixed-use residential zones where small-cell density is increasing. We need to update the standard installation kit to include RF shielding as a baseline requirement for District 7 deployments."
